CSS CHEAT SHEET

1. SELECTORS

Selectors are used to target the HTML elements on the web page we want to style. CSS selector is the first part of a CSS Rule.

PROPERTIES
      Attribute:- This is used to select an element with some specific attribut or value.
      Class:- This is a way to select all of the elements with the specified class name and must start with a period (.)
      Type:- This is a way of matching elements with the corresponding element node name such as paragraph,div and span tags
      IDs:- This is a way to select only the element with the specified id and must start with a pound sign(#)

2. VISUAL RULES

Theses are used to add styles to web pages such as the sizes, colors, e.t.c.

PROPERTIES
      Font-family:- This defines the typeface of an element
      Font-size:- This controls the size of the text displayed
      Font-weight:- This defines how thin or thick text is displayed
      Text-align:- This places text in the right, left, center of its parent container
      Color:- This defines the color of the text(foreground color)
      Background-color:- This defines the color behind the text
      Opacity/Alpha:- This defines the tranparency of a text
      Background-image:- This places an image behind text
      Mark:- This is used to highlight text
      Pre:- This is used to preserve all spaces and linebreks

3. THE BOX MODEL

This is used when talking about design and layout.

PROPERTIES
      Height and Width:- This sets the content area of textx in pixels(px) or percentage(%)
      Borders:- These surround the content area and padding of an element
      Padding:- This is the space between the content area and the borders
      Margin:- This is the amount of spacing outside an element's border
      Horizontal margin:- This adds so the total space between the borders of adjacent elements is equal to the sum of the right margin of one element and the left margin of the adjacent element
      Vertical margin:- This collapse so the space between vertically adjacent elements are equal to the larger margin
      Margin: 0 auto:- This horizontally centers an element inside of its parent content area if t has a width
      Overflow:- This can be set to display, hide, scroll, and dictates how HTML will render content that overflows its parent content area
      Visibility:- This can hide or show elements

4. POSITIONING/ LAYOUT

This specifies the position of an element

PROPERTIES
      Absolute:- This is positioned relative to its closest parent element
      Fixed:- This position is pinned to any part of the web page, the elements will remainin view no matter what
      Sticky:- This makes an element stick to a defined offset position when its parent container is been scrolled
      Z-index:- This specifies how far back or how far forward an element appears on the page when it oerlap other elements
      Display:- This controls how an element flows vertially and horizontally in a document
      Inline:- This takes up as little space as possible but they cannot have manually adjusted width or height
      Block:- This takes up the width of their container and can have manually adjusted width
      Inline-Block:- This can have a set width and height
      Float:- This can move elements as far left or as far right as possible on a web page
      Clear:- This is use to clear an element left or right sides or both
